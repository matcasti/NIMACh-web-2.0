HRV Analyzer — Polar H10 Web App

Aplicación web para análisis de variabilidad de frecuencia cardíaca en tiempo real usando bandas Polar H10 vía Web Bluetooth. Sin instalación, sin cuenta, sin costo.

✓ Métricas en dominio tiempo (SDNN, RMSSD, pNN50)
✓ Análisis frecuencial (LF, HF, LF/HF)
✓ Poincaré plot en tiempo real
✓ Exportación CSV / JSON
✓ Compatible con Polar H10 y H9
Web Bluetooth JavaScript HRV Polar H10 Open Source

Nuestro compromiso

Diversidad, Equidad e Inclusión

Creemos que la ciencia es más poderosa cuando refleja la diversidad de quienes la protagonizan. La equidad no es un objetivo separado — es parte constitutiva de nuestra misión como grupo.

⚖️

Equidad en ciencia

Promovemos oportunidades equitativas de formación y participación, con foco en grupos históricamente subrepresentados en ciencias de la salud.

🌍

Diversidad cultural

Colaboraciones con Brasil, España, México y toda Latinoamérica enriquecen nuestros enfoques metodológicos y las preguntas de investigación.

🤝

Inclusión activa

Vinculamos la ciencia con estudiantes de enseñanza media de Magallanes, democratizando el acceso al conocimiento científico desde el fin del mundo.
